About

Introduction

This website allows visitors to create an account, sign in, Sign out, and delete their account. Once they have created an account, they can search for recipes and generate meal plans for the week or for the day. The website uses the Spoonacular API to integrate with Spoonacular's recipe database.

Features

The website uses the following features:

  • User registration and authentication (password hashing)
  • Recipe Search
  • Meal plan generation
  • Spoonacular API integration

Technology Stack

The website was built with the following technologies:

  • Flask: A Python web framework
  • WTForms: A Python form library
  • Python: Back-end development
  • jQuery: A JavaScript library
  • PostgreSQL: A relational database management system (RDBMS)
  • JavaScript: Connect front-end with back-end
  • HTML & CSS: Front-end development
  • SQLAlchemy: SQL toolkit for Python
  • Jinja2: Template engine for Python
